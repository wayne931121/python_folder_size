# python_folder_size
你可以使用這個程式檢視電腦上的資料夾和檔案大小。

[Google Translate]: You can use these codes to view the folders and files size on your Window(or others platform).

## 安裝需求(install dependent):
```
pip install -r requirements.txt
```

## 執行程式(Run Code):
```
python size.py
```
